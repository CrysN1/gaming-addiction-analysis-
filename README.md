# 🎮 Gaming Addiction Analysis

## 📌 Sobre o Projeto

Projeto desenvolvido para praticar técnicas de Análise de Dados utilizando Excel, SQL e Power BI.

O objetivo foi entender padrões relacionados ao vício em jogos através de análises exploratórias, limpeza dos dados e construção de dashboards.

---

## Dataset

- 250 registros
- 49 colunas
- Sem valores nulos
- Sem registros duplicados

---

## Principais Insights

- Homens possuem maior índice de dependência.
- Jogadores de PC apresentaram maiores níveis de estresse.
- A faixa entre 18 e 28 anos concentra os maiores índices.
- MMORPG é o gênero mais jogado.
- Jogadores que trabalham costumam possuir mais anos de experiência.
